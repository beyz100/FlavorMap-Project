About FlavorMap

A restaurant discovery app built for CSE 220 Web Programming.

About This Project

FlavorMap was developed as part of the CSE 220 Web Programming course. The goal of this project is to create a restaurant discovery platform where users can explore restaurants, read and write reviews, manage favorites, and view menus and opening hours.


Built with Django.
